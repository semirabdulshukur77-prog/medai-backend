# 🚀 MedAI Backend Setup Guide

This guide provides step-by-step instructions to set up the MedAI backend server for development and production environments.

## 📋 Prerequisites

### Required Software
- **Python 3.8 or higher** ([Download](https://www.python.org/downloads/))
- **pip** (Python package manager, comes with Python)
- **Virtual Environment** (venv or conda)
- **GROQ API Key** ([Get free key here](https://console.groq.com/keys))

### Optional (for production)
- **PostgreSQL** (recommended for production)
- **Redis** (for caching)
- **Docker** (for containerization)
